# Introduction

Welcome to the **jEAP** (Java Enterprise Application Platform) documentation.

## What is jEAP?

jEAP is a collection of shared libraries, Spring Boot starters, and tools that provide a standardized foundation for building enterprise Java applications. It addresses common cross-cutting concerns such as:

- **Messaging** – Kafka-based messaging with inbox/outbox patterns
- **Security** – OAuth2/OIDC integration and authorization
- **Monitoring** – Metrics, tracing, and health checks
- **Testing** – Test utilities and integration test support

Explore the documentation to learn more about jEAP's capabilities and how to integrate them into your projects.
